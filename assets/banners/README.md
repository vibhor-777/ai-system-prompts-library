# 🖼️ Banners

This directory contains banner images and assets for the AI System Prompts Library.

## Contents

| File | Description |
|------|-------------|
| `hero-banner.png` | Main repository hero banner |
| `og-image.png` | Open Graph social preview image |
| `dark-banner.png` | Dark theme banner variant |
| `light-banner.png` | Light theme banner variant |

## Usage

Reference these banners in the README or documentation:

```markdown
![Hero Banner](assets/banners/hero-banner.png)
```

## Guidelines

- All banners should be in PNG or SVG format
- Recommended resolution: 1280×640px for hero banners
- OG images should be 1200×630px
- Keep file sizes under 500KB
