# Safe Mode — Configuration

## Overview

The Safe Mode configuration applies maximum safety guardrails, appropriate for consumer-facing products, educational platforms, children's applications, and any context where vulnerable users may be present.

| Setting | Value |
|---------|-------|
| Safety level | Maximum |
| Content policy | Strictest defaults |
| Audience | General public / potentially vulnerable users |
| Crisis handling | Enhanced safe messaging |
| Refusal style | Clear, kind, with alternatives |

## Key Behaviors

- Enhanced safe messaging for mental health, self-harm, and crisis topics
- Conservative content thresholds across all categories
- Proactive professional referrals for serious matters
- Age-appropriate language and content by default
- No engagement with dual-use topics even with stated professional context
- Explicit recommendation to verify all health and safety information
